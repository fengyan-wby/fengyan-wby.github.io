## 截图工具flameshot

```bash
sudo apt install flameshot
```

设置快捷键：
设置 > 键盘快捷键 > 自定义快捷键 > +
名称: flameshot
命令: /usr/bin/flameshot gui

## FTP工具FileZilla

```bash
sudo apt-get install filezilla
```

## Terminator终端模拟器

```bash
sudo apt-get install terminator
```

## 文本编辑Sublime Text

```bash
sudo apt-get install sublime-text
```
